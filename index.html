04 ผู้ช่วยสรุปข้อมูลด้วย AI

สรุปสถานการณ์อัตโนมัติ

วิเคราะห์ข้อมูลตามตัวกรองที่เลือกอยู่ในขณะนี้ พร้อมข้อเสนอแนะเชิงนโยบาย

ทำไมบางครั้งสรุป AI ใช้ไม่ได้? ปุ่มนี้เรียกใช้ Claude โดยตรง ซึ่งทำงานได้ทันทีแบบไม่ต้องตั้งค่าเมื่อเปิดไฟล์นี้ ภายใน Claude.ai (เช่นหน้าต่างตัวอย่างไฟล์) แต่เมื่อดาวน์โหลดไฟล์มาเปิดเองหรือนำไปโฮสต์ที่อื่น จะไม่มีสิทธิ์เรียก API โดยอัตโนมัติ (ขึ้น "Failed to fetch")

วิธีแก้สำหรับการใช้งานนอก Claude.ai: ใส่ Anthropic API key ของคุณเอง (สร้างได้ที่ console.anthropic.com) คีย์จะถูกเก็บไว้ในเบราว์เซอร์นี้เท่านั้น (localStorage) และถูกส่งตรงจากเบราว์เซอร์ไปยัง Anthropic โดยไม่ผ่านเซิร์ฟเวอร์อื่น — อย่าใช้คีย์นี้บนเครื่องสาธารณะ/ที่ใช้ร่วมกับผู้อื่น เพราะใครก็ตามที่เปิด DevTools ของเบราว์เซอร์นี้จะเห็นคีย์ได้
กดปุ่ม "สรุปข้อมูลด้วย AI" เพื่อให้ผู้ช่วยวิเคราะห์ข้อมูลตามตัวกรองปัจจุบัน และอธิบายภาพรวมเป็นภาษาที่เข้าใจง่าย

แหล่งข้อมูล: ไฟล์ข้อมูลแหล่งกำเนิดมลพิษ กทม. + สคพ. (การเชื่อมต่อระบบบำบัดน้ำเสียรวม) — บันทึกล่าสุดจากไฟล์ต้นทาง 7 ก.ค. 2569. แดชบอร์ดนี้ประมวลผลทั้งหมดในเบราว์เซอร์ของคุณ ไม่มีการส่งข้อมูลออกนอกเครื่อง (ยกเว้นการเรียกสรุปด้วย AI และการดึงข้อมูลจาก Google Sheet)
หมายเหตุ: ข้อมูลเป็นภาพนิ่งที่ฝังในไฟล์นี้ กด "เชื่อมต่อ Google Sheet" มุมขวาบนเพื่อดึงข้อมูลฉบับล่าสุดได้ทุกที่ทุกเวลา โดยไม่ต้องอัปโหลดไฟล์เอง
